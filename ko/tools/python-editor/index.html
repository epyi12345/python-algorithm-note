    파이썬 에디터

    로컬 IDE 또는 Python 실행 환경이 없을 때 이 임베디드 에디터를 사용하세요.

    민감한 데이터를 붙여넣지 마세요

    private code, API keys, tokens, passwords, 민감한 데이터는 온라인 에디터에 붙여넣지 마세요. 이 에디터는 외부 서비스에서 제공됩니다.

    Embedded Editor

    사용 방법

    • Python solution 코드를 에디터에 붙여넣습니다.
    • stdin 또는 input 영역에 sample input을 붙여넣습니다.
    • 코드를 실행하고 expected output과 비교합니다.
    • 최종 정답 코드는 problems/custom/에 저장합니다.
    • 시도 과정과 피드백은 docs/wrong-notes/에 기록합니다.
