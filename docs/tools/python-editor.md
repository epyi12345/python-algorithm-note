is not available.

## Usage Notes

- Paste your Python solution into the editor.
- Paste sample input into the stdin or input area.
- Run the code and compare the output with the expected output.
- For final records, save the accepted solution in problems/custom/.
- Record attempts and feedback in docs/wrong-notes/.
