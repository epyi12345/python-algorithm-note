## 사용 방법

- Python solution 코드를 에디터에 붙여넣습니다.
- stdin 또는 input 영역에 sample input을 붙여넣습니다.
- 코드를 실행하고 expected output과 비교합니다.
- 최종 정답 코드는 `problems/custom/`에 저장합니다.
- 시도 과정과 피드백은 `docs/wrong-notes/`에 기록합니다.
