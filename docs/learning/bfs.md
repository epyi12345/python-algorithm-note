# BFS

## What it is
Breadth-first search explores by distance layers.

## When to use it
- Shortest path in unweighted graph/grid

## Basic pattern
Queue start node, pop, expand neighbors, mark visited.

## Small example
Grid shortest path from S to E.

## Common mistakes
- Missing visited
- DFS used for shortest-path guarantee
