# 동률 정렬

## 개념

동률 처리(tie-break)는 1차 값이 같은 항목들 사이의 출력 순서를
정하는 규칙입니다.

## 언제 쓰나

- 같은 점수/수량을 가진 항목이 여러 개일 때
- 출력 순서를 안정적으로 고정해야 할 때

## 기본 문법

```python
# 동률 최대값 항목을 이름 순으로 정렬
for name in sorted(candidates):
    print(name, candidates[name])
```

## 작은 예시

- 값 기준 정렬은 큰 값을 먼저 찾는 데 유용합니다.
- 이름 기준 정렬은 동률 항목의 순서를 결정합니다.

## 자주 하는 실수

- 값 기준 정렬만 하고 동률 규칙을 누락하는 경우
- 동률 항목을 입력 순서대로 출력하는 경우

## 안전한 패턴

- 먼저 목표 값(예: 최댓값) 계산
- 이후 동률 key를 요구된 기준으로 정렬
