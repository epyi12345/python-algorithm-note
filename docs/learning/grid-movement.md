# Grid Movement

## What it is
Systematic movement on row/column coordinates with direction vectors.

## When to use it
- Maze/grid traversal
- BFS/DFS on 2D map

## Basic pattern
Use direction arrays and boundary checks before moving.

## Small example
`dirs = [(1,0),(-1,0),(0,1),(0,-1)]`

## Common mistakes
- Row/column swap
- Missing boundary or wall checks
