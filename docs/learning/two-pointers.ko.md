# 투 포인터(two pointers)

## 이 개념은 무엇인가
투 포인터(two pointers)는 왼쪽 포인터와 오른쪽 포인터를 이동시키며 연속 구간을 효율적으로 탐색하는 기법입니다.

## 언제 사용하는가
- 연속 구간의 합/길이/조건 만족 여부를 찾을 때
- 한 번 순회에 가깝게 문제를 풀고 싶을 때
- 특히 양수 배열에서 구간 합 조건을 다룰 때

## 핵심 아이디어
- 오른쪽 포인터를 늘려 구간을 확장하고, 조건을 만족하면 왼쪽 포인터를 이동해 구간을 축소합니다.
- 양수 배열에서는 오른쪽을 늘리면 합이 커지고, 왼쪽을 줄이면 합이 작아진다는 성질을 활용합니다.
- 이 성질 덕분에 불필요한 재탐색 없이 연속 구간을 처리할 수 있습니다.

## 기본 문법 또는 패턴
```python
left = 0
window_sum = 0
for right in range(n):
    window_sum += arr[right]
    while window_sum >= target:
        # 조건 만족 시 정답 갱신
        window_sum -= arr[left]
        left += 1
```

## 단계별 예시
1. 오른쪽 포인터를 이동해 연속 구간에 새 값을 추가합니다.
2. 조건(예: 구간 합 >= target)을 만족하면 왼쪽 포인터를 이동해 더 짧은 구간을 시도합니다.
3. 다시 조건이 깨질 때까지 축소를 반복합니다.
4. 이 과정을 전체 배열에 대해 진행하면 각 포인터는 한 방향으로만 이동합니다.
5. `0004. Target Sales Window` 같은 문제에서 최소 길이/조건 만족 구간 탐색에 자주 사용됩니다.

## 흔한 실수
- 양수 가정이 필요한 패턴을 음수 포함 배열에 그대로 적용하는 실수
- `while` 안에서 포인터나 합을 갱신하지 않아 무한 루프가 생기는 실수
- 연속 구간 문제인데 비연속 원소 선택으로 해석하는 실수

## 안전한 패턴
- 먼저 "연속 구간" 문제인지 확인합니다.
- 양수 배열 가정이 필요한지 문제 조건을 확인합니다.
- 루프마다 포인터 이동이 보장되도록 코드를 작성합니다.
- 디버깅 시 `(left, right, window_sum)`를 출력해 흐름을 검증합니다.

## 시간복잡도
- 왼쪽/오른쪽 포인터가 각각 최대 `N`번 이동하므로 보통 `O(N)`입니다.
