# 투 포인터(two pointers)

## 개념
두 개의 인덱스로 구간을 표현하고 규칙에 따라 이동시키는 기법입니다.

## 언제 쓰나
- 연속 구간 문제
- 양수 배열 구간 합 문제

## 기본 패턴
오른쪽 확장으로 구간을 키우고, 조건을 만족하면 왼쪽 축소로 최적화합니다.

## 작은 예시
양수 배열에서 합이 S 이상인 최소 길이 구간 찾기.

## 자주 하는 실수
- 비연속 선택
- 왼쪽 축소 누락
