# 학습 자료

이 섹션은 저장소를 단순 문제 보관소가 아닌, 짧은 복습 중심의
알고리즘 학습 가이드로 활용하기 위해 구성되었습니다.

## 모바일에서 사용하는 방법

- 한 번에 한 페이지씩 짧게 읽기
- 3~7분 단위로 집중 복습하기
- 페이지마다 핵심 포인트 1개 기록하기
- 익숙해질 때까지 반복해서 다시 보기

## 추천 출퇴근 학습 흐름

1. 패턴 1개 복습
2. 자주 하는 실수 1개 읽기
3. 이미 푼 문제 1개 다시 보기
4. 리뷰 체크리스트 1개 점검
