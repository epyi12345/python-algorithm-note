# 단조 스택(monotonic stack)

## 개념
스택의 정렬 성질(단조 증가/감소)을 유지하며 한 번의 순회로 문제를 푸는 기법입니다.

## 언제 쓰나
- 다음 더 큰 값
- 이전 더 작은 값

## 기본 패턴
현재 값이 단조 조건을 깨면 pop하며 해당 인덱스를 해결합니다.

## 작은 예시
다음 성장일까지 대기 일수 계산 O(N).

## 자주 하는 실수
- 엄격/비엄격 비교 혼동
- 미해결 기본값 처리 누락
