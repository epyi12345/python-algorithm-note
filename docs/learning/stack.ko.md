# 스택(stack)

## 이 개념은 무엇인가
스택(stack)은 LIFO(Last-In, First-Out) 방식으로 동작하는 자료구조입니다.

## 언제 사용하는가
- 괄호 짝 맞추기
- 최근 작업 취소(undo)
- “가장 최근 미해결 항목”을 추적하는 문제
- 단조 스택(monotonic stack) 기반 응용 문제

## 핵심 아이디어
가장 나중에 넣은 값을 가장 먼저 꺼냅니다.
파이썬에서는 보통 리스트의 `append`, `pop`으로 구현합니다.

## 기본 문법 또는 패턴
```python
stack=[]
for ch in s:
    if ch=="(": stack.append(ch)
    elif stack: stack.pop()
```

## 단계별 예시
1. 문자열을 왼쪽에서 오른쪽으로 순회합니다.
2. 여는 괄호를 만나면 스택(stack)에 넣습니다.
3. 닫는 괄호를 만나면 최근 여는 괄호를 꺼내 매칭합니다.
4. 순회 종료 후 스택이 비어 있으면 균형 상태입니다.

## 흔한 실수
- 비어 있는 스택에서 `pop()`을 호출하는 실수
- 거리/인덱스가 필요한 문제에서 값만 저장하는 실수
- 스택의 의미(“미해결 상태”)를 중간에 바꿔버리는 실수

## 안전한 패턴
- 꺼내기 전 비어 있는지 검사
- 인덱스가 필요하면 스택에 인덱스를 저장
- 스택 원소가 무엇을 의미하는지(미해결 항목) 주석으로 고정

## 시간복잡도
각 원소는 최대 한 번 push, 한 번 pop 되므로 전체 O(N)입니다.
