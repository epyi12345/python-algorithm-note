# Problem Solving Patterns

| Signal in Problem | Possible Pattern |
|---|---|
| Duplicate handling | set |
| Counting frequency | dict / Counter |
| Product or user aggregation | dict / defaultdict |
| Range sum queries | prefix sum |
| Consecutive interval | two pointers |
| Recent item / undo | stack |
| Waiting line / process order | queue |
| Shortest movement count | BFS |
| Connected components | DFS / BFS |
| Minimum or maximum possible value | binary search |
| Repeated optimal choice | greedy |
| Previous result affects next result | DP |
