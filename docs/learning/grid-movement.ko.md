# 격자 이동

## 개념
행/열 좌표와 방향 벡터로 2차원 격자를 체계적으로 탐색하는 방법입니다.

## 언제 쓰나
- 미로/격자 탐색
- BFS/DFS 2차원 탐색

## 기본 패턴
방향 배열을 사용하고, 이동 전에 경계/벽 검사를 수행합니다.

## 작은 예시
`dirs = [(1,0),(-1,0),(0,1),(0,-1)]`

## 자주 하는 실수
- 행/열 인덱스 혼동
- 경계/벽 검사 누락
