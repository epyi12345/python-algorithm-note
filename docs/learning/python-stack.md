# Python Stack

## What it is
LIFO structure typically implemented with list append/pop.

## When to use it
- Undo/backtracking states
- Next greater/smaller style scans

## Basic pattern
`stack.append(x)` and `stack.pop()` while condition holds.

## Small example
Keep unresolved indexes and resolve when a larger value appears.

## Common mistakes
- Popping from empty stack
- Storing values when indexes are needed
