# Learning Materials

This section turns the repository into a compact algorithm study guide,
not only a problem archive. Use it to review core concepts, avoid repeat
mistakes, and prepare quick problem-solving routines.

## How to use on mobile

- Open one short page at a time.
- Focus on 3 to 7 minute reading blocks.
- Save one key takeaway per page.
- Revisit the same page until the pattern feels automatic.

## Recommended commute study flow

1. Review one pattern.
2. Read one common mistake.
3. Revisit one solved problem.
4. Check one review checklist.
