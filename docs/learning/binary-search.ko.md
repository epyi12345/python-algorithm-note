# 이분 탐색(binary search)

## 개념
단조 조건이 있는 탐색 공간을 절반씩 줄여 답을 찾는 기법입니다.

## 언제 쓰나
- 정렬 데이터 탐색
- 정답 공간 탐색

## 기본 패턴
mid가 가능하면 오른쪽 경계를 줄이고, 불가능하면 왼쪽 경계를 늘립니다.

## 작은 예시
[최대 패키지, 전체 합] 범위에서 최소 배송 용량 찾기.

## 자주 하는 실수
- 경계값 설정 오류
- 경계 갱신 실수로 무한 루프
