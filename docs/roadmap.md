# Training Roadmap

Use this roadmap to turn daily practice into measurable improvement.

## Stage 1: Python Problem-Solving Basics

Goal: become comfortable translating requirements into Python code.

- Practice input parsing and output formatting.
- Review lists, dictionaries, sets, tuples, and strings.
- Write simple helper functions for repeated logic.
- Estimate time complexity for loops and nested loops.

## Stage 2: Core Coding Test Patterns

Goal: recognize common patterns quickly from constraints and keywords.

- Hash tables for counting, lookup, and grouping.
- Sorting for ordering, ranking, and interval-style problems.
- Prefix sums for repeated range queries.
- Two pointers for sorted arrays and contiguous ranges.
- Stack and queue usage for simulation and traversal.
- BFS/DFS for graph, grid, and connected-component problems.
- Greedy and binary search for decision and optimization problems.
- Heap for priority-based retrieval.

## Stage 3: DP and Advanced Implementation

Goal: solve problems that require state definition and careful transitions.

- Define DP states in words before coding.
- Identify base cases and transition formulas.
- Compare recursive memoization and iterative tabulation.
- Practice advanced implementation with simulation and combined patterns.

## Daily Routine

1. Solve one timed problem.
2. Record blocked points.
3. Rewrite the accepted solution.
4. Solve one similar problem.
